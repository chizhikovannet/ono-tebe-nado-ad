# Оно тебе надо — аукцион вещей, в которые никто не верил

Проектная работа для Яндекс Практикума. Одностраничный сайт аукциона необычных вещей.

## О проекте

Верстка выполнена по макету в Figma. Сайт состоит из пяти основных блоков:
- Шапка с навигацией
- Обложка с заголовком и кнопкой
- Секция с карточками лотов
- Секция с информацией об аукционе
- Подвал с контактами и соцсетями

## Технологии

- HTML5
- CSS3
- Flexbox
- Grid Layout

## Особенности реализации

- Сетка построена на Grid Layout и Flexbox
- Адаптивная верстка для ноутбуков (ширина 1100px)
- Переиспользуемые классы для оверлея и адреса
- Семантическая верстка с использованием `<header>`, `<main>`, `<section>`, `<article>`, `<address>`
